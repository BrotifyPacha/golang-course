# Задание 2

Добавляем локализацию на русский язык.

1. Добавляем отказоустойчивость
    1. Ручка `/set` должна сохранять переданные данные в файл `data.txt`.
    1. При запуске приложения проверяем есть ли в текущей директории файл
    `data.txt`, и если есть, берём в качестве значения по умолчание его
    содержимое.
        * В случае когда есть файл и cli аргументы, приоритет должено иметь значение
        переданное через cli аргументы.

1. Добавляем локализацию
    1. Добавляем локализацию вывода ручки `/`
        * При запросе проверяем содержимое `Accept-Language`, и если оно равно `ru-RU`,
        то выводим соответствующий текст
    1. Ручка `/set` должна тепреь работать с json телом, которое должно иметь вид:

    ```json
    {
        "eng": "hello world!",
        "ru": "привет мир!"
    }
    ```
